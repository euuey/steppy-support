개인정보처리방침

시행일자: 2026년 3월 1일  ·  최종 업데이트: 2026년 3월 1일

Steppy(이하 "서비스")는 이용자의 개인정보를 중요하게 생각하며, 관련 법령 및 서비스 운영 원칙에 따라 개인정보를 안전하고 투명하게 처리하고자 합니다. 본 개인정보처리방침은 Steppy가 어떤 정보를 수집하고, 어떤 목적으로 이용하며, 어떻게 보관 및 삭제하는지 설명합니다.

1. 적용 대상
📱 적용 범위

본 개인정보처리방침은 Android 앱 Steppy 및 이와 관련된 서비스에 적용됩니다.

2. 수집하는 정보
🗂️ 수집 항목

Steppy는 서비스 제공을 위해 다음과 같은 정보를 수집하거나 처리할 수 있습니다.

2-1. 계정 및 식별 정보
  • Google Sign-In을 통해 제공되는 계정 정보
  • 이메일 주소
  • 사용자 ID
  • 닉네임 또는 프로필 정보
2-2. 피트니스 정보
  • 걸음 수 데이터
  • 활동량 관련 데이터
2-3. 서비스 이용 정보
  • 게임 방 정보 및 게임 진행 상태
  • 전적 정보 (예: 승/패 기록)
  • 친구 요청 및 친구 관계 데이터
  • 랭킹 표시를 위한 통계 정보
  • 계정 삭제 또는 데이터 삭제 요청 처리 정보
3. 개인정보의 이용 목적
🎯 이용 목적

Steppy는 수집한 정보를 다음 목적을 위해서만 이용합니다.

  • 로그인 및 사용자 계정 식별
  • 만보기 기능 제공
  • 걸음 수 기반 게임 진행 및 보상 계산
  • 친구 기능 제공 및 멀티플레이 게임 진행
  • 전적 및 랭킹 표시
  • 계정 관리 및 데이터 삭제 요청 처리
  • 서비스 운영, 안정성 유지 및 보안 대응
4. 개인정보의 제공 및 외부 서비스 이용
🔗 외부 서비스
✅ Steppy는 이용자의 개인정보를 판매하지 않습니다.

서비스 제공을 위해 아래 외부 서비스를 사용할 수 있으며, Steppy는 서비스 제공에 필요한 범위 내에서만 이를 이용합니다.

Google Google Sign-In 및 관련 인증 기능
Supabase 사용자 계정, 게임 데이터, 전적, 친구 데이터 등 백엔드 처리

각 외부 서비스의 개인정보 처리에 대해서는 해당 서비스의 개인정보처리방침을 참고하시기 바랍니다.

5. 개인정보의 보관 및 삭제
🗄️ 데이터 보관 정책

Steppy는 서비스 제공에 필요한 기간 동안 개인정보를 보관하며, 이용자의 요청 또는 처리 목적 달성 시 지체 없이 삭제합니다.

5-1. 계정 삭제 시 삭제되는 데이터

이용자가 계정 삭제를 요청할 경우 다음 데이터가 삭제됩니다.

데이터 유형 처리 추가 보관
계정 정보 삭제 없음
프로필 정보 삭제 없음
게임 진행 데이터 삭제 없음
전적 데이터 삭제 없음
친구 관련 데이터 삭제 없음
5-2. 계정을 유지한 채 삭제할 수 있는 데이터

이용자는 계정 삭제 없이 일부 데이터 삭제를 요청할 수 있습니다. 예를 들어 다음과 같은 데이터 삭제를 요청할 수 있습니다.

  • 게임 통계 또는 전적 데이터 초기화
  • 친구 관련 데이터 삭제
5-3. 별도로 보관되는 데이터
없음 — Steppy는 계정 삭제 요청 처리 후 별도로 보관하는 데이터가 없습니다.
5-4. 추가 보관 기간
없음 — 삭제 요청이 처리되면 추가 보관 기간 없이 즉시 삭제됩니다.
6. 이용자의 권리
👤 이용자 권리

이용자는 언제든지 다음을 요청할 수 있습니다.

  • 계정 삭제 요청
  • 계정을 유지한 채 일부 데이터 삭제 요청
  • 개인정보 처리 관련 문의
7. 계정 삭제 및 데이터 삭제 요청 방법
🗑️ 삭제 요청 방법

이용자는 아래 방법을 통해 계정 삭제 또는 데이터 삭제를 요청할 수 있습니다.

  • 앱 내 회원탈퇴 또는 데이터 관리 기능 사용
  • 아래 문의처를 통한 이메일 요청
  • 계정 삭제 / 데이터 삭제 안내 페이지 이용: https://your-github-pages-url
요청 시 필수 제공 정보: Steppy에 사용 중인 Google 계정 이메일 주소를 함께 제공해야 할 수 있습니다.
8. 보안
🔒 보안 조치

Steppy는 이용자의 개인정보를 보호하기 위해 합리적인 기술적·관리적 보안 조치를 적용합니다. 다만, 인터넷 환경의 특성상 완전한 보안을 절대적으로 보장할 수는 없습니다. Steppy는 보안 취약점 발견 시 신속하게 대응하고자 합니다.

9. 아동의 개인정보
👦 아동 개인정보 보호

Steppy는 관련 법령에 따라 필요한 경우를 제외하고 아동의 개인정보를 고의로 수집하지 않습니다. 아동의 개인정보가 수집된 사실을 인지하는 경우, 즉시 삭제하기 위한 조치를 취할 것입니다.
